# Specifications

## Technical Specs

- Frontend: Vite React TypeScript
- CSS Framework: Tailwind CSS

## Context

- การเลือกตั้งทั่วไป 2569 จะเกิดขึ้นในวันที่ 8 กุมภาพันธ์ 2569 การเลือกตั้งล่วงหน้าและนอกเขต จะเกิดขึ้นในวันที่ 1 กุมภาพันธ์ 2569
- สำหรับการประชามติจะเกิดขึ้นในวันที่ 8 กุมภาพันธ์ 2569 แต่จะไม่มีการเลือกตั้งล่วงหน้า จะมีเพียงการเลือกตั้งนอกเขตเท่านั้น โดยจะเกิดขึ้นในวันที่ 8 กุมภาพันธ์ 2569 พร้อมกันทั่วประเทศ
- ถ้าหากประชาชนอยู่นอกพื้นที่ในวันที่ 8 กุมภาพันธ์ 2569 จะต้องไปเลือกตั้งนอกเขตในวันที่ 1 กุมภาพันธ์ 2569 แต่ยังต้องไปออกเสียงประชามติในวันที่ 8 กุมภาพันธ์ 2569 อีกครั้งหนึ่ง

## Features

- สอบถามผู้ใช้งานไปทีละขั้นตอน
- ขั้นแรก ถามผู้ใช้ว่า มีสิทธิเลือกตั้งอยู่ในพื้นที่จังหวัดใด (พร้อมแสดงลิงก์ตรวจสอบสิทธิ์)
- ถัดมา ถามว่า ในวันที่ 8 กุมภาพันธ์ คุณจะอยู่ที่จังหวัดใด มีตัวเลือกว่า อยู่ในจังหวัดเดียวกับจังหวัดที่มีสิทธิ หรือว่าจังหวัดอื่น ถ้าตอบจังหวัดอื่น ให้เลือกจังหวัด
- ถัดมา หากผู้ใช้ตอบว่าอยู่ในจังหวัดอื่น ให้ถามว่า ในวันที่ 1 กุมภาพันธ์ คุณอยู่ที่จังหวัดใด มีตัวเลือกว่า อยู่ในจังหวัดเดียวกับจังหวัดที่มีสิทธิ หรือจังหวัดเดียวกับที่อยู่วันที่ 8 กุมภาพันธ์ 2569 หรือว่าจังหวัดอื่น ถ้าตอบจังหวัดอื่น ให้เลือกจังหวัด
- แสดงผลลัพธ์ว่า ทั้งการเลือกตั้ง และการประชามติ คุณควรจะลงทะเบียนเลือกตั้งล่วงหน้า นอกเขต หรือไม่ ที่จังหวัดใด และไปเลือกตั้งในวันไหนบ้าง พร้อมแสดงลิงก์เพื่อลงทะเบียนเลือกตั้งล่วงหน้า นอกเขต โดยแสดงผลทั้งหมดในหน้าจอเดียว เพื่อให้สะดวกต่อการบันทึกภาพหน้าจอเก็บไว้
